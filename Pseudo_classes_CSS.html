What are Pseudo-classes?

A pseudo-class is used to define a special state of an element.
For example, it can be used to:

  • Style an element when a user mouses over it
  • Style an element when it gets focus
  • Style visited and unvisited links differently

Syntax for pseudo classes:

selector:pseudo-class {
        property: value; 
}
Click me

The CSS used to make it:

/* unvisited link */
    a:link {
    color: #FF0000;
}

/* visited link */
    a:visited {
    color: #00FF00;
}

/* mouse over link */
a:hover {
    color: #FF00FF;
}

/* selected link */
a:active {
    color: #0000FF;
}

:lang() pseudo-class allows you to define special rule for different languages.

Use of :focus pseudo-class

On clicking the input field as we know the white outline appears showing that it is focused. But, Clicking the input field above will make a red outline. The reason for that is the use of :focus pseudo-class.

The CSS for it:

input:focus{
    outline:none;
    border:2px solid red;
}
