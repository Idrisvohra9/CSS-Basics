CSS Flexbox Layout Module

Before the Flexbox Layout module, there were four layout modes:

  • Block, for sections in a webpage
  • Inline, for text
  • Table, for two-dimensional table data
  • Positioned, for explicit position of an element

A Flexible Layout must have a parent element with the display property set to flex.

The flex container properties are:

  • flex-wrap
  • flex-direction
  • flex-flow
  • justify-content
  • align-items
  • align-content

There are flex item properties in CSS which we will not be going in detail coz they are fairly straightforward and simple.

The list of the flex item properties and its descriptions:

Property Description
align-self Specifies the alignment for a flex item (overrides the flex container's align-items property)
flex-grow Specifies how much a flex item will grow relative to the rest of the flex items inside the same container
flex-shrink Specifies how much a flex item will shrink relative to the rest of the flex items inside the same container
flex-basis Specifies the initial length of a flex item
order Specifies the order of the flex items inside the same container
Last notes:
The elements of a container with display flex are treated same no matter if they are div or span if the css property is set the same in both cases. So block or inline elements are treated as flex elements once the container is set to display flex.
