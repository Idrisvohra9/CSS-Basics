Grid Lines

Grid lines are a way to see the grid item start and end points which will come handy later on for spanning the items inside a grid container.

Column span or grid column

item1
2
3
4
5
6
item7

This can be achieved by the css property:

                    .item1{
                    grid-column: 1 / 3;
                    }

                    or

                    .item1{
                    grid-column-start: 1;
                    grid-column-end: 3;
                    }

item 7 css:

                    .item7 {
                    grid-column: 2 / 4;
                    }
